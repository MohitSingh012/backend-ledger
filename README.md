# Backend Ledger

A Node.js backend application built using Express.js and MongoDB.

## Features

- User Authentication
- Email Verification with Nodemailer
- JWT Authentication
- MongoDB Database
- Secure Environment Variables

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Nodemailer
- JWT
